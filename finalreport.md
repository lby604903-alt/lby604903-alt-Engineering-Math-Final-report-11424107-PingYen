## Step 1: Establish the Differential Equation

The relationship between the input signal and the output voltage is

```math
RC\frac{dy(t)}{dt}+y(t)=f(t)
```

The input signal is

```math
f(t)=\left[\sin(10t)+0.5\sin(1000t)\right]u(t)
```

The initial condition is

```math
y(0)=0
```
